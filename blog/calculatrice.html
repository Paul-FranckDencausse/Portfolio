Créer une Calculatrice Simple en .NET

Un projet idéal pour débuter avec le développement d'applications en .NET

Présentation du projet

La Calculatrice Simple en .NET est une application de base qui permet d’effectuer les opérations mathématiques fondamentales : addition, soustraction, multiplication et division. Ce projet est parfait pour s’initier à la programmation en .NET, que ce soit en mode console ou avec une interface graphique Windows Forms.

Fonctionnalités principales

  • Additionner deux nombres
  • Soustraire deux nombres
  • Multiplier deux nombres
  • Diviser deux nombres (avec gestion de la division par zéro)

Prérequis

  • Visual Studio (édition Community ou supérieure)
  • .NET SDK installé sur votre machine

Installation rapide

  1. Cloner le dépôt : 
    git clone https://github.com/votre-utilisateur/calculatrice-simple.git
  2. Ouvrir le projet :
    • Lancez Visual Studio
    • Sélectionnez "Ouvrir un projet ou une solution"
    • Ouvrez le fichier .sln du dépôt cloné

Utilisation

Application Console
  1. Appuyez sur F5 ou cliquez sur "Démarrer" dans Visual Studio pour lancer l’application.
  2. Suivez les instructions à l’écran pour entrer les nombres et choisir l’opération.
  3. Le résultat s’affiche directement dans la console.
Application Windows Forms (optionnel)
  1. Lancez l’application comme précédemment.
  2. Entrez les nombres dans les champs prévus.
  3. Sélectionnez l’opération dans le menu déroulant.
  4. Cliquez sur "Calculer" pour obtenir le résultat.

Contribuer au projet

Les contributions sont les bienvenues ! Pour participer :

  1. Forkez le projet
  2. Créez une branche pour votre fonctionnalité : 
    git checkout -b feature/nouvelle-fonctionnalite
  3. Commitez vos modifications : 
    git commit -m 'Ajouter une nouvelle fonctionnalité'
  4. Poussez vers la branche : 
    git push origin feature/nouvelle-fonctionnalite
  5. Ouvrez une Pull Request
