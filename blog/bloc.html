Article de bloc

D'un côté j'aime Minecraft (j'ai même le livre de cuisine officiel) de l'autre j'ai envie d'apprendre Java. Alors autant faire d'une pierre, deux coups!

Fonctionnalités

  • Ajoute un nouveau bloc avec une texture . Pour la personnalisation voyez ça avec un vrai artiste.
  • Le bloc peut être placé et cassé dans le monde de Minecraft.

Prérequis

  • Minecraft Forge installé pour la version correspondante.
  • Java Development Kit (JDK) installé.

Installation

Téléchargez le Mod :

Téléchargez le fichier .jar du mod depuis la section des releases.

Installer le Mod :

Placez le fichier .jar dans le dossier mods de votre installation Minecraft.

Lancer Minecraft :

Lancez Minecraft avec le profil Forge pour charger le mod.

Utilisation

  • Créez un Nouveau Monde ou Chargez un Monde Existant : le nouveau bloc sera disponible dans l'inventaire créatif.
  • Placez le Bloc : sélectionnez-le et placez-le comme n’importe quel autre bloc.

Personnalisation

  • Texture : modifiez my_first_block.png dans src/main/resources/assets/mymod/textures/blocks/.
  • Propriétés : ajustez dureté, résistance, etc. dans MyFirstBlock.java.

Développement

Préparer l'environnement

  • Installez le JDK.
  • Installez Minecraft Forge MDK pour la version souhaitée.
  • Configurez votre IDE (IntelliJ IDEA ou Eclipse).

Construire le Mod

  • Générez le projet : 
    gradlew setupDecompWorkspace eclipse  # Pour Eclipse
gradlew setupDecompWorkspace idea    # Pour IntelliJ IDEA
  • Construisez le mod : 
    gradlew build
  • Testez le mod depuis votre IDE.

Contribuer

  • Forkez le projet.
  • Créez une branche pour votre fonctionnalité : git checkout -b feature/nouvelle-fonctionnalite
  • Commitez vos modifications : git commit -m 'Ajouter une nouvelle fonctionnalité'
  • Poussez vers la branche : git push origin feature/nouvelle-fonctionnalite
  • Ouvrez une Pull Request.
