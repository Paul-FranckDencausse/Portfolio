Créer un deck Anki Français-Portugais facilement avec Python

Automatisez la création de vos cartes de vocabulaire pour progresser en portugais européen !

Pourquoi ce projet ?

Apprendre une langue, c’est avant tout mémoriser du vocabulaire. Anki est l’outil idéal pour cela, mais saisir chaque carte à la main peut vite devenir fastidieux. Ce projet vous permet de générer automatiquement un deck Anki à partir d’un simple fichier texte, pour accélérer votre apprentissage du portugais européen.

Fonctionnement

  1. Préparez un fichier vocab.txt contenant vos paires de mots, une par ligne, au format :
    mot_francais : mot_portugais
  2. Lancez le script Python fourni (vocab_anki.py) pour transformer ce fichier en un CSV compatible Anki.
  3. Importez le fichier CSV dans Anki et commencez à réviser !

Installation rapide

  1. Clonez le dépôt : 
    git clone https://github.com/ton-repository/vocab-fr-pt-anki.git
cd vocab-fr-pt-anki
  2. Créez votre fichier vocab.txt avec vos mots.
  3. Générez le CSV : 
    python vocab_anki.py

Importation dans Anki

  1. Ouvrez Anki et cliquez sur "Importer".
  2. Sélectionnez le fichier fr_pt_deck.csv généré.
  3. Choisissez la virgule comme séparateur de champs.
  4. Assurez-vous que les champs correspondent à Recto/Verso.
  5. Cliquez sur "Importer" et le tour est joué !

Personnalisation

Vous souhaitez inverser les langues (portugais → français) ou ajouter plus de mots ? Il suffit de modifier le fichier vocab.txt ou d’adapter le script Python pour changer l’ordre des colonnes.

Veillez à respecter le format mot_francais : mot_portugais pour chaque ligne.

Conclusion

Ce projet vous fait gagner un temps précieux et vous permet de vous concentrer sur l’essentiel : apprendre et pratiquer le portugais européen. N’hésitez pas à contribuer ou à adapter le script à vos besoins !
